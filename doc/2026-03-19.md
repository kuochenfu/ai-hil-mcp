### Serial MCP Server (`serial-mcp/server.py`)
- Built with FastMCP 3.1.1, Python 3.14, pyserial 3.5
- Tools implemented:
  - `list_serial_ports` — lists all connected serial devices
  - `read_serial_log(port, baud, lines)` — reads N lines, auto-flags anomalies (`HardFault`, `Panic`, `Watchdog`, `Assert`, `Stack Overflow`)
  - `send_serial_command(port, command, baud, response_lines)` — sends command and captures response
- Registered in Claude Code via `.mcp.json` (project-scoped)
- Smoke tested: `list_serial_ports` detected ST-Link V3 at `/dev/cu.usbmodem1303`

### Project Setup
- Created `setup.sh` — run once in any new project to register all MCP servers

## Decisions
- **Option A (project-scoped MCP)** chosen over user-scoped: keeps each project's MCP config explicit and version-controlled in `.mcp.json`
- **FastMCP over raw MCP SDK**: less boilerplate, decorator-based, hot-reload dev mode
- Building all MCP servers ourselves — no existing hardware/embedded MCP servers found in the ecosystem
