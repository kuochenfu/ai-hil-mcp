## Decisions
- **Option A (project-scoped MCP)** chosen over user-scoped: keeps each project's MCP config explicit and version-controlled in `.mcp.json`
- **FastMCP over raw MCP SDK**: less boilerplate, decorator-based, hot-reload dev mode
- Building all MCP servers ourselves — no existing hardware/embedded MCP servers found in the ecosystem

### JTAG MCP Server (`jtag-mcp/server.py`)
